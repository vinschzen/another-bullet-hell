## Installation

1. Clone the repository:

    ```bash
    $ git clone hhttps://github.com/vinschzen/another-bullet-hell.git
    $ cd project
    ```
2. Set up Unity Hub

    ```bash
    $ Projects -> Add -> Select %YourPC%/another-bullet-hell
    $ Install editor version 2022.3.10f1 LTS
    ```
